*Cryptio test*

how to run app :

build docker image :
`$> docker build -t cryptio_test .`

run container :
`$> docker-compose up -d`

go to -> http://localhost:5000

-----------------------------------------------

dev : 

tests :
`$> npm run test`

linter + nodemon :
`$> npm run start-dev`

build react :
`(/client)$> npm run build`
